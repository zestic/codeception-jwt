# Codeception JWT
#### A codeception extension for asserting JWT

#### Installation
```bash
composer require zestic/codeception-zestic --dev
```

#### To configure

In your acceptance.suite.yml file
```yaml
modules:
    enabled:
        - JWT:
            algorithm:
            publicKeyPath:
            privateKeyPath:
            tokenTtl: 
```
